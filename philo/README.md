## ⚙️ Compilação

 - Para compilar o projeto, execute:

       make

 - Para limpar os arquivos objeto:

       make clean

 - Para remover tudo (obj e executável):

       make fclean

## ▶️ Execução

Após compilar, execute o programa com os parâmetros (n_philosophers time_to_die time_to_eat time_to_sleep e number_of_times_each_philosopher_must_eat(opcional)):

    ./philo 5 800 200 200
